Triggers are used to start rules and have a type and a value. Triggers can be re-used.

Trigger types

There are 3 kinds of triggers:

  1. cronjob
  2. event
  3. manual

The cronjob trigger type accepts values in standard crontab syntax (e.g. *****) and requires setting up the scheduler.

The event trigger type accepts the name of the event (e.g. eloquent.saved: {Author}{Plugin}\Models{Model} or rainlab.user.register).

The manual trigger type requires no value
